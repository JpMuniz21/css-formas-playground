🎨 Desafio: Formas Geométricas com HTML e CSSProjeto desenvolvido para a disciplina de Desenvolvimento de Plataformas Web na Universidade de Fortaleza (UNIFOR).

📌 Objetivo: O objetivo principal deste desafio foi criar diversas formas geométricas utilizando estritamente HTML5 e CSS3, sem o uso de JavaScript ou imagens externas.

🛠️ Tecnologias e Propriedades ExploradasPara a construção das formas, foram aplicadas as seguintes propriedades CSS:Dimensionamento: width e height.Estilização: background e border.Formatos: border-radius para círculos e ovais.Manipulação: transform (rotações) e position (posicionamento absoluto).

🚀 O que foi desenvolvido?O projeto está dividido em quatro partes principais:Formas Básicas: Quadrado, Retângulo, Círculo e Oval.Nível Médio (Bordas): Triângulos (cima e baixo) e Trapézio.Formas Avançadas: Estrela, Losango e Coração.Bônus Extra: Criação de um Emoji, uma Casa composta e animações com @keyframes.
